# 🐍 Snake

A classic Snake game built from scratch with plain HTML, CSS, and JavaScript — no frameworks, no libraries. Fully responsive canvas, on-screen directional controls, keyboard support, and a pulsing food animation.

## 🎮 Features

- Smooth grid-based movement with collision detection (walls + self)
- Snake head with directional eyes that track movement
- Pulsing food animation
- Live score tracking
- On-screen ▲ ◀ ▼ ▶ controls for mobile/touch play
- Full keyboard arrow key support
- Restart button + Game Over state
- Fully responsive canvas — fits any screen without scrolling

## 🕹️ How to Play

- **Desktop:** Use the Arrow Keys to move.
- **Mobile/Touch:** Tap the on-screen ▲ ◀ ▼ ▶ buttons.
- Eat the red food to grow and score points.
- Avoid hitting the walls or yourself.
- Hit **Restart** to play again after Game Over.

## 🛠️ Built With

- HTML5 Canvas
- CSS3
- Vanilla JavaScript (no frameworks or dependencies)
